Bio

I started learning Java in school circa 2009 and since then greatly expanded my knowledge in my own time, pretty much exclusively off the Internet.

I’ve managed to create a 2D game engine called Grame (both for fun and as a means of teaching myself by overcoming obstacles).

I’ve taught myself (what I like to think is) all there is to know about the Swing Java GUI library and have created a few programs which demonstrate a nice blend of responsiveness, good looks and functionality. I’ve included a few of those here on my site; kindly refer to Reaction Keeper and Tracking Manager.

Projects

In development

  • Grame - A 2D grid based game engine. Has been my main focus for a few years now and I still don’t think it’s finished.

  • MooCommands - A command handling and processing library. Great for things like IRC clients.

  • MooConsole - A shell console that lets you run MooCommands commands.

  • Mootilities - A “swiss army knife” for Java. Contains a ton of help classes which I’m sure you’ll find handy.

  • MooBootstrap - An update detection and handling library. Essentially an auto updater for your program.

Pretty much finished™

  • Reaction Keeper - A super intuitive tag based index for all your reaction images/GIFs. Need a meme? Never get caught with your pants down.

  • Tracking Manager - A simple tool that helps you keep track of your packages. Great if you’re an artist that finds themselves sending a lot of work to clients using services like UPS, USPS, FedEx, etc.
